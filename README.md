## Autonomous launch loop

1. **Intent capture:** translate rough operator energy into a shippable repo scope.
2. **Skill loading:** use GitHub, image, planning, and verification workflows.
3. **Research pass:** source-ground the rave history and DJ terminology.
4. **Asset generation:** create a hero image that tells the story visually.
5. **Repo build:** write structured docs, data, image, and verifier.
6. **Verification:** run local checks and git hygiene.
7. **Publish:** create public GitHub repo and verify remote main.
8. **Reveal packaging:** create this second repo so humans can see the launch method.
