### What is Sonic-Forage?

An open-source, forkable ecosystem for the music scene — combining AI music creation, DJ tools, rave culture history, harm reduction, and community building into one autonomous system.

### What's in it?

**14 repos. 13 live websites. 345+ files. 11 AI-generated hero images. All built autonomously.**

- 🎵 **Autonomous DJ OS** — AI-powered DJ companion with Unicode CDJ, companion pets, and offline mixing
- 📖 **Rave Culture Field Guide** — Full uncensored history from disco to Eclipse 2045
- 🛡️ **Harm Reduction** — Community care built into everything
- 🎛️ **OSC Command Panel** — Trippy expanding control surface for live performance
- 🎪 **Festival Organizer** — Event planning, gig finder, booking toolkit
- 🎨 **ComfyUI Integration** — AI music and image generation
- 📱 **Mobile Kiosk** — Festival installation concept
- 🌑 **Eclipse 2045 Roadmap** — 24-hour autonomous DJ marathon on my birthday

### How to get started

```bash
git clone https://github.com/Sonic-Forage/sonic-forage-starter-kit.git
cd sonic-forage-starter-kit
bash install.sh
```

The installer will:
1. Show you a birth animation
2. Ask you to name your digital entity
3. Choose a personality
4. Create your workspace
5. Install Hermes Agent (the AI brain)
6. Load the rave culture guide
7. Connect harm reduction resources
8. Guide you to the next steps
